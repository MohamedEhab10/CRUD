# CRUD Application using JavaScript

## Description
This is a simple **CRUD (Create, Read, Update, Delete)** application built using **JavaScript, HTML, and CSS**.  
It allows users to manage data dynamically in the browser, such as creating, viewing, updating, and deleting items.  
The application is lightweight, easy to understand, and perfect for beginners learning DOM manipulation and basic JavaScript logic.

## Features
- **Create**: Add new items to the list.
- **Read**: View all existing items.
- **Update**: Edit existing items.
- **Delete**: Remove items from the list.
- **Dynamic UI**: Changes are immediately reflected without refreshing the page.
- **LocalStorage (Optional)**: Persist data across browser sessions.
